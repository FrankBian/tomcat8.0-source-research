org.apache.tomcat.util.digester

Class CallParamRule

  • java.lang.Object
    • org.apache.tomcat.util.digester.Rule
      • org.apache.tomcat.util.digester.CallParamRule

  • public class CallParamRule
extends Rule

    Rule implementation that saves a parameter for use by a surrounding CallMethodRule.

    This parameter may be:

    • from an attribute of the current element See CallParamRule(int paramIndex, String attributeName)
    • from current the element body See CallParamRule(int paramIndex)
