org.apache.catalina.comet

Interface CometProcessor

  • All Superinterfaces:
    Servlet

    public interface CometProcessor
extends Servlet
    This interface should be implemented by servlets which would like to handle asynchronous IO, receiving events when data is available for reading, and being able to output data without the need for being invoked by the container. Note: When this interface is implemented, the service method of the servlet will never be called, and will be replaced with a begin event.
