Interface ActionHook

  • All Known Implementing Classes:
    AbstractAjpProcessor, AbstractHttp11Processor, AbstractProcessor, AjpAprProcessor, AjpNio2Processor, AjpNioProcessor, AjpProcessor, Http11AprProcessor, Http11Nio2Processor, Http11NioProcessor, Http11Processor, SpdyProcessor

    public interface ActionHook
    Action hook. Actions represent the callback mechanism used by coyote servlet containers to request operations on the coyote connectors. Some standard actions are defined in ActionCode, however custom actions are permitted. The param object can be used to pass and return informations related with the action. This interface is typically implemented by ProtocolHandlers, and the param is usually a Request or Response object.
